# 📝 Hipotesa

Platform jurnal untuk mendokumentasikan hipotesa dan riset.

## ✨ Fitur

- **Topik** — Filsafat, Ekonomi, Bisnis, Agama, Sains, Teknologi, Politik, Seni, Kesehatan, Pendidikan, Kehidupan, Parenting
- **Hipotesa** — Dokumentasi pertanyaan riset dengan struktur lengkap
- **Argumen** — Bagian pendukung hipotesa
- **Kesimpulan** — Hasil dan kesimpulan
- **Sumber** — Daftar reading list dengan status dan URL
- **Updates** — Log perkembangan jurnal
- **Tema** — Light/Dark mode
- **Export/Import** — Backup data dalam format JSON

## 🚀 Deploy

Live: https://hipotesa-journal-production.up.railway.app/

## 💻 Development

```bash
# Install dependencies
npm install
cd server && npm install && cd ..

# Run dev server
npm run dev
```

## 🛠️ Tech Stack

- React + Vite
- Tailwind CSS
- Express.js
- PostgreSQL (Railway)
